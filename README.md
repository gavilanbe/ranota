# 🐸 RaNota

Un juego 3D voxel donde eres una rana en un mundo cúbico generado por procedimientos. Salta entre voxels, usa tu lengua para cazar moscas y explora un mundo verde renderizado en 3D con Babylon.js.

## ✨ Características

- 🧊 Mundo voxel generado por procedimientos con ruido (noise) y mallas optimizadas (greedy meshing).
- 👅 Mecánica de lengua para atrapar moscas a distancia.
- 🪰 Moscas con comportamiento propio para cazar.
- 🐸 Control de rana con física de salto en primera/tercera persona.
- ✨ Efectos visuales (VFX), audio y UI integrados.
- 🌿 Ambiente verde con iluminación 3D real.

## 🚀 Cómo jugar / ejecutar

Necesitas Node.js. Instala dependencias y arranca el servidor de desarrollo:

```
npm install
npm run dev
```

Para generar la build de producción y previsualizarla:

```
npm run build
npm run preview
```

## 🎮 Controles

- 🖱️ Haz clic en "Click to Play" para capturar el ratón y mirar alrededor.
- ⌨️ Teclas de movimiento + salto para desplazar a la rana.
- 👅 Acción de lengua para cazar moscas.

## 🛠️ Tecnología

- [Babylon.js](https://www.babylonjs.com/) (`@babylonjs/core`, `gui`, `materials`) — motor 3D.
- [Vite](https://vitejs.dev/) — bundler y dev server.
- JavaScript con módulos ES (target esnext).
